# Surprisal-measurements
This repository contains the surprisal measurements for a number of .wav files, recorded for the project 'Comprehension under Pressure', conducted by Petter Kallioinen and others at Stockholm University. Moreover, it contains the transcriptions of the .wav-files and Python scripts to sentence tokenize the transcriptions and to calculate the surprisal values for every word on a sentence level.  
## Installation
To install the surprisal package in surprisal_calculator.py, see https://github.com/aalok-sathe/surprisal.
